# json

python json handle library

## Used by pyenv virtualenv json

    pyenv virtualenv  3.10.12 json
    pyenv activate json
    python -m pip install --upgrade pip
    cd util/json
    pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
